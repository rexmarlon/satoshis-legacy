#  SATOSHIS LEGACY - Projektplan

## Übersicht

**Projektziel:** Neue Blockchain basierend auf Bitcoin Core v30.2 (Januar 2026).

**Motto:** "Be Early Again" - Moderner Code, frischer Start, Early Adopter Feeling.

---

## Technische Spezifikationen

| Parameter | Wert |
|-----------|------|
| **Codebasis** | Bitcoin Core v30.2 |
| **Magic Bytes** | `0x534c4547` (SLEG) |
| **Mainnet Port** | 9333 |
| **Block Time** | 10 Minuten |
| **Block Reward** | 50 SLEG |
| **Max Supply** | 21.000.000 SLEG |
| **Algorithm** | SHA-256 (PoW) |
